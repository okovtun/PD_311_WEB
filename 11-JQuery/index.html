$(selector).action(); $(this).hide() - позволяет спрятать текущий элемент; $("p") - спрятать все абзацы;

Selectors

  • "*" - все элементы;
  • this - означает текущий элемент;
  • "ul li:first" - спрятать первый элемент списка;
  • "ul li:parent" - спрятать родительский элемент;
  • "[attrib]" - спрятать все элементы с заданным атрибутом;
  • ":button" - скрывает все элементы типа button и <input type="button">

JQuery Á DOM

  • Get/Set:
    • text() - возвращает текстовое содержимое элемента;
    • html() - возвращает все содержимое элемента, включая HTML-разметку;
    • val() - возвращает значения полей;
  • Add:
    • append() - добавляет текст после содержимого элемента (перед закрывающим декстриптором);
    • prepend() - добавляет текст перед содержимым элемента (после открывающего дескриптора);
    • before() - добавляет текст перед открывающим дескриптором;
    • after() - добавляет текст после закрывающего дескриптора;
  • Remove:
    • remove() - удаляет сам выбранный жоемент;
    • empty() - удаляет все дочерние элементы из выбранного;

JQuery ♥ CSS

  • addClass() - добавляет один и более классов для выбранного элемента;
  • removeClass() - удаляет один и более классов для выбранного элемента;
  • toggleClass() - при последоательных вызовах добавляет/удаляет класс к выбранному элементу;
  • css() - чтение/запись атриьута 'style' выбранного элемента;
