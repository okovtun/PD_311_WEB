DataTypes

  1. Логические типы:
    • Boolean (bool)
  2. Числовые типы:
    • Number - это дробное число, размером 8 Byte (double). Любое число по умолчанию является Number.;
    • Bigint - это целое число, поддерживающее длинную арифметику. Значение типа BigInt можно создать при помощи конструктора класса BigInt(), либо же после указанного числового значения поставить 'n';
  3. Символьные типы:
    • Symbol
  4. Ссылочные типы:
    • String
    • Object
    • Null
  5. Undefined

Operators

  1. Unary:
    1. Arithmetical:+, -
    2. Increment/Decrement (++/--)
  2. Binary:
    1. Arithmetical:+,-,*,/,%,** - оператор возведения в степень
    2. Составные присваивания: +=, -=, *=, /=, %=, **=
    3. Операторы сравнения:
      • ==
      • !=
      • >
      • <
      • >=
      • <=
      • === - строгое равно
      • !== - строгое НЕравно
      Строгие сравнения кроме значений так же сравнивают еще и типы.
  3. Ternary: condition ? value_1 : value_2
